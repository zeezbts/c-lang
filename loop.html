Looping Statements in C

In C programming, loops are used to repeat a specific block of code until a condition is met. This helps in reducing code length and saving time.

1. The For Loop

Best used when the number of iterations is known in advance.

// Syntax: for(initialization; condition; increment/decrement)
for(int i = 1; i <= 5; i++) {
  printf("%d ", i);
}
Output: 1 2 3 4 5

2. The While Loop

An entry-controlled loop that repeats as long as the condition remains true.

int i = 1;
while(i <= 3) {
  printf("Zeez ");
  i++;
}
Output: Zeez Zeez Zeez

3. The Do-While Loop

An exit-controlled loop that always executes the code at least once.

int i = 10;
do {
  printf("Value is %d", i);
  i++;
} while(i < 5); // Condition is false, but it runs once
Output: Value is 10

Important Points to Remember

  • Entry Controlled Loops: for and while check the condition before running the code.
  • Exit Controlled Loop: do-while checks the condition after running the code.
  • Infinite Loop: A loop that never ends, e.g., while(1) { ... }.
  • Break Statement: Immediately exits the loop.
  • Continue Statement: Skips the current turn and jumps to the next iteration.
Exam Tip: Use a "do-while" loop when the program must execute the code block at least once (like a Menu-driven program).
